# cpp-maze-map-generator

### Symbols:
- █  **→** wall/block
- \* **→** Bomb
- P  **→** Player
- E  **→** End Point
- \+ **→** correct path
  
### Notes:
- comment Line ```456``` for removing bombs and bombs within the map
- for showing the path pass ```true``` to ```maze.printField()```
